### Use shadow

- Şuan 25 seviyeye kadar otomatik gölgelendirmeyi `shadow={0...24}` aralığında girerek yapabilirsiniz.
- Yakın zamanda gölge oluşturucu entegre edilerek teorik olarak sonsuz gölgelendirmeyi projeye kazandıracağız.
- Ayrıca Gölge oluşturucu ile yaklaşık 800byte alan kazanmamızı da sağlayacaktır

```js live=true column=true
<div style={{overflowX:"hidden",maxWidth:"40vw"}}>
  <div
  style={{
    display: "flex",
    flexWrap: "wrap",
    alignItems: "center",
    justifyContent: "center",
    overflowX:"overlay"
  }}
>
  {new Array(6).fill("").map((_, shadowLevel) => (
    <Card
      shadow={shadowLevel}
      width={120}
      height={120}
      p={34}
      m={34}
      key={shadowLevel}
    >
      <Text>Level:{shadowLevel}</Text>
    </Card>
  ))}
</div>
</div>
```
